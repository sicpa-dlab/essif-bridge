# OpenID Connect - SIOP rest API

## Description

Rest API created with [Nest](https://github.com/nestjs/nest) framework that:
- sign and verify JWT
- issue credential to Validated ID mobile wallet.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
