# eSSIF Bridge API

[OpenAPI 3](https://github.com/OAI/OpenAPI-Specification) service specification.


## Setup

```
npm install
```

## How to use

### dev

Run http server for development

```
npm run dev
```

### validate

Validates `openapi.yaml`

```
npm run validate
```

### bundle

Generate a single OpenAPI yaml file: `build/openapi.yaml`
```
npm run bundle
```
